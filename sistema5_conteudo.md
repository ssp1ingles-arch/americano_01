## O QUE É SHADOWING

Shadowing = você vê a frase em português, tenta dizer em inglês
antes de ver a resposta, depois repete em voz alta 3 vezes.

É a técnica mais eficaz para desenvolver fluência automática —
você para de "traduzir na cabeça" e começa a falar diretamente.

COMO USAR ESTE SISTEMA:
1. Veja a frase em português
2. Tente dizer em inglês em voz alta (sem ver a resposta)
3. Veja a frase em inglês
4. Repita 3 vezes em voz alta
5. Avance para a próxima

### GRUPO 1 — ESTADOS (I'm feeling... / I feel...)
*Como você está agora — 15 frases*

```
01. Estou me sentindo pronto.      → I feel ready.
02. Estou me sentindo ótimo.       → I'm feeling great.
03. Estou me sentindo calmo.       → I'm feeling calm.
04. Estou me sentindo forte.       → I'm feeling strong.
05. Estou me sentindo cansado.     → I'm feeling tired.
06. Estou me sentindo bem.         → I'm feeling good.
07. Estou me sentindo melhor.      → I'm feeling better.
08. Estou me sentindo pronto agora.→ I feel ready now.
09. Estou bem.                     → I'm doing fine.
10. Estou indo bem.                → I'm doing okay.
11. Estou melhorando.              → I'm doing better.
12. Estou fazendo o meu melhor.    → I'm doing my best.
13. Ainda não estou pronto.        → I'm not ready yet.
14. Estou quase pronto.            → I'm almost ready.
15. Estou me sentindo confiante.   → I'm feeling confident.
```

**PADRÃO DO GRUPO:**
- I feel + [adjetivo] → estado permanente ou mais formal
- I'm feeling + [adjetivo] → estado no momento atual
- I'm doing + [adjetivo/advérbio] → como você está se saindo

**MACETE:**
"How are you?" → "I'm doing great!" (não apenas "I'm great")
"I'm doing" soa mais natural e fluente.

```
87.  Soa familiar.                 → It sounds familiar.
88.  Parece importante.            → It seems important.
89.  Parece interessante.          → It looks interesting.

```

**PADRÃO DO GRUPO — OS 4 SENTIDOS:**
- It looks = parece (visualmente)
- It sounds = soa / parece (auditivamente ou por informação)
- It feels = parece (sensação / intuição)
- It seems = parece (opinião / impressão geral)

**MACETE VISUAL:**
```
OLHOS  → looks  → "It looks good" (você vê)
OUVIDOS→ sounds → "It sounds nice" (você ouve ou ouviu falar)
TOQUE  → feels  → "It feels right" (você sente / intuição)
MENTE  → seems  → "It seems hard" (você acha / impressão)
```

### GRUPO 7 — ISSO/THIS (This is... / This feels...)
*Falando de algo próximo — 15 frases*

```
91.  Isso parece bom.              → This feels good.
92.  Isso parece bonito.           → This looks beautiful.
93.  Isso soa ótimo.               → This sounds great.
94.  Isso parece certo.            → This feels right.
95.  Isso funciona bem.            → This works well.
96.  Isso me ajuda.                → This helps me.
97.  Isso é suficiente.            → This is enough.
98.  Isso é perfeito.              → This is perfect.
99.  Isso é melhor.                → This is better.
100. Isso é simples.               → This is simple.
101. Isso é diferente.             → This is different.
102. Isso faz sentido.             → This makes sense.
103. Isso é o que eu quero.        → This is what I want.
104. Isso é incrível.              → This is amazing.
105. Isso muda tudo.               → This changes everything.
```

**PADRÃO DO GRUPO:**
- This = isso/este (perto de você, específico)
- That = aquilo/aquele (longe de você ou já mencionado)
- "This is what I want" → This como sujeito complexo

**MACETE:**
This = aqui perto, agora
That = lá longe, antes, ou abstrato

"This coffee is great." (o café que está na minha mão)
"That coffee we had yesterday was great." (o de ontem)

```
106. Aquilo parece bom.            → That feels good.
107. Aquilo parece bonito.         → That looks beautiful.
108. Aquilo soa ótimo.             → That sounds great.
109. Aquilo parece certo.          → That feels right.
110. Aquilo funciona bem.          → That works well.
111. Aquilo me ajuda.              → That helps me.
112. Aquilo é suficiente.          → That is enough.
113. Aquilo é perfeito.            → That is perfect.
114. Aquilo é melhor.              → That is better.
115. Aquilo é simples.             → That is simple.
116. Aquilo foi incrível.          → That was amazing.
117. Aquilo faz sentido.           → That makes sense.
118. Aquilo é o que eu quero.      → That's what I want.
119. Aquilo mudou tudo.            → That changed everything.
120. Aquilo foi perfeito.          → That was perfect.
```

**COMPARAÇÃO THIS vs THAT:**
```
This is good.  → Isso (na sua mão) é bom.
That is good.  → Aquilo (lá) é bom.
That was good. → Aquilo foi bom. (passado)
That sounds good. → Isso parece bom. (sobre ideia/plano)
```

**MACETE:**
"That sounds good!" → usado para concordar com planos
"That makes sense!" → usado para confirmar compreensão
"That's what I want!" → ênfase no desejo

### GRUPO 9 — VAMOS (Let's...)
*Convidando para ação — 15 frases*

```
121. Vamos tentar novamente.       → Let's try again.
122. Vamos começar agora.          → Let's start now.
123. Vamos seguir em frente.       → Let's move on.
124. Vamos embora agora.           → Let's go now.
125. Vamos continuar.              → Let's keep going.
126. Vamos manter a calma.         → Let's stay calm.
127. Vamos manter o foco.          → Let's stay focused.
128. Vamos fazer uma pausa.        → Let's take a break.
129. Vamos terminar isto.          → Let's finish this.
130. Vamos fazer melhor.           → Let's do better.
131. Vamos trabalhar juntos.       → Let's work together.
132. Vamos pensar nisso.           → Let's think about this.
133. Vamos resolver isso.          → Let's figure this out.
134. Vamos ser honestos.           → Let's be honest.
135. Vamos aproveitar.             → Let's enjoy this.
```

**PADRÃO DO GRUPO:**
- Let's + [verbo base] = vamos + [verbo]
- Let's = Let us (forma contraída)
- Sempre seguido de verbo NO INFINITIVO SEM "to"

**ERRO CLÁSSICO:**
❌ "Let's to go" → ✓ "Let's go"
❌ "Let's to try" → ✓ "Let's try"

**EXPRESSÕES ESPECIAIS:**
- "Let's move on" = vamos seguir / passar para o próximo assunto
- "Let's keep going" = vamos continuar / não pare
- "Let's figure this out" = vamos resolver isso / descobrir

**WILL vs GOING TO:**
- I will = decisão no momento / promessa
- I'm going to = plano já feito

"Oh, the phone's ringing — I'll get it!" (decisão agora)
"I'm going to call you tomorrow." (plano já feito)
