## PILAR 2 — 1.000 FRASES DO INGLÊS REAL
### Organizado do básico ao avançado

---

### NÍVEL 1 — SOBREVIVÊNCIA (frases 1–100)

#### Bloco A — Sim, Não, Não sei (1–15)
```
1.  Yes. / Yeah. / Yep.
    → Sim. (formal → informal → muito informal)
    FALADO: "yeah" e "yep" são os mais usados. "Yes" soa formal demais no dia a dia.

2.  No. / Nope. / No way.
    → Não. / De jeito nenhum.
    FALADO: "nope" é casual, "no way" é ênfase — "De forma alguma!"

3.  I don't know. / I have no idea. / No clue.
    → Eu não sei. / Não faço ideia.
    FALADO: "I dunno" (I don't know contraído)

4.  Maybe. / Probably. / I think so.
    → Talvez. / Provavelmente. / Acho que sim.

5.  Of course. / Sure. / Absolutely.
    → Claro. / Com certeza.
    FALADO: "sure" é o mais natural no dia a dia americano

6.  I'm not sure. / I'm not sure about that.
    → Não tenho certeza.

7.  It depends. / Depends on the situation.
    → Depende. / Depende da situação.
    MACETE: brasileiros traduzem "depende" mas esquecem — em inglês é "it depends", não só "depends"

8.  Really? / Are you serious? / No way!
    → Sério? / Não acredito!

9.  Exactly. / That's right. / Correct.
    → Exatamente. / Isso mesmo.

10. Not really. / Not exactly. / Kind of.
    → Na verdade não. / Mais ou menos.
    FALADO: "kinda" = kind of

11. I guess so. / I suppose so.
    → Suponho que sim. / Acho que sim.

12. We'll see. / Let's see.
    → A gente vê. / Vamos ver.

13. Who knows? / Hard to say.
    → Quem sabe? / Difícil dizer.

14. Fine. / Okay. / Alright.
    → Tudo bem. / Ok.
    FALADO: "alright" = "aight" em fala muito informal

15. Whatever. / Doesn't matter.
    → Tanto faz. / Não importa.
```

#### Bloco B — Pedidos básicos (16–30)
```
16. Can I have...? / I'd like...
    → Posso ter...? / Eu gostaria de...
    EXEMPLO: "Can I have a coffee, please?"

17. Can you help me? / I need help.
    → Você pode me ajudar? / Preciso de ajuda.

18. Where is the bathroom? / Where's the restroom?
    → Onde fica o banheiro?
    MACETE: "restroom" é mais educado que "bathroom" em público

19. How much does it cost? / How much is it?
    → Quanto custa?

20. I need... / I want... / I'd like...
    → Preciso de... / Quero... / Gostaria de...
    MACETE: "I'd like" é mais educado que "I want" em lojas e restaurantes

21. Can I get...? / Let me get...
    → Posso pegar...?
    FALADO: "Let me get a coffee" — fala natural nos EUA

22. Do you have...? / Is there...?
    → Vocês têm...? / Tem...?

23. I'm looking for...
    → Estou procurando...

24. Can you repeat that? / Sorry?
    → Pode repetir? / Como?
    FALADO: "Sorry?" com entonação de pergunta = "Como?" em inglês britânico

25. Can you speak slowly, please?
    → Pode falar devagar, por favor?

26. I don't understand. / I didn't catch that.
    → Não entendi. / Não peguei.

27. What does ___ mean?
    → O que significa ___?

28. How do you say ___ in English?
    → Como se diz ___ em inglês?

29. I'm lost. / Can you show me on the map?
    → Estou perdido. / Pode me mostrar no mapa?

30. Call the police! / Call an ambulance!
    → Chame a polícia! / Chame uma ambulância!
```

#### Bloco C — Apresentações e saudações (31–50)
```
31. Hi! / Hey! / Hello!
    → Oi! / Olá!
    FALADO: "hey" é o mais comum entre amigos

32. How are you? / How's it going? / What's up?
    → Como você está? / E aí?
    FALADO: "what's up?" = "wassup?" / resposta: "not much"

33. I'm fine, thanks. / Pretty good. / Not bad.
    → Estou bem, obrigado. / Muito bem. / Nada mal.

34. My name is ___. / I'm ___.
    → Meu nome é ___. / Eu sou ___.
    FALADO: "I'm Tom" é mais natural que "My name is Tom"

35. Nice to meet you. / Good to meet you.
    → Prazer em conhecer você.

36. Where are you from? / I'm from Brazil.
    → De onde você é? / Sou do Brasil.

37. What do you do? / I'm a ___.
    → O que você faz? / Sou ___.
    MACETE: "What do you do?" = profissão, não atividade no momento

38. How old are you? / I'm ___ years old.
    → Quantos anos você tem?

39. Do you speak English? / A little.
    → Você fala inglês? / Um pouco.

40. I'm learning English.
    → Estou aprendendo inglês.

41. My English isn't perfect.
    → Meu inglês não é perfeito.

42. Please be patient with me.
    → Por favor tenha paciência comigo.

43. See you later. / See you tomorrow. / Bye!
    → Até mais. / Até amanhã. / Tchau!
    FALADO: "see ya" = see you

44. Take care. / Have a good one.
    → Cuida-se. / Tenha um bom dia.

45. Good morning / afternoon / evening / night.
    → Bom dia / boa tarde / boa noite (chegando) / boa noite (saindo)

46. Welcome! / Come in!
    → Bem-vindo! / Entre!

47. Make yourself at home.
    → Fique à vontade.

48. Can I introduce you to ___?
    → Posso te apresentar a ___?

49. We've met before, right?
    → Já nos conhecemos, certo?

50. I've heard a lot about you.
    → Já ouvi muito sobre você.
```
