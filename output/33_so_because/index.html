SO · BECAUSE
Dois lados da mesma história — consequência e causa
SO = então / tão BECAUSE = porque SO THAT = para que
role para baixo
Conceito central
A mesma situação — duas perspectivas
Situação: choveu → fiquei em casa
SO — você narra o EFEITO
It was raining, so I stayed home.
Estava chovendo, então fiquei em casa.
chuva → resultado: fiquei
BECAUSE — você narra a CAUSA
I stayed home because it was raining.
Fiquei em casa porque estava chovendo.
fiquei ← razão: chuva
mesma situação · perspectiva diferente · ambas corretas
Conectivo 1
SO — Então / Tão / Para que
1
SO = então / portanto — resultado/consequência
A segunda parte é o que acontece por causa da primeira
TipoInglêsPortuguês
resultadoIt was raining, so I stayed home.Estava chovendo, então fiquei em casa.
resultadoI was hungry, so I ate.Eu estava com fome, então comi.
resultadoShe studied hard, so she passed.Ela estudou muito, então passou.
resultadoHe forgot his wallet, so I paid.Ele esqueceu a carteira, então eu paguei.
futuroI'll be busy tomorrow, so call today.Vou estar ocupado amanhã, então ligue hoje.

Use vírgula antes de SO quando une duas orações completas: "It rained, so I stayed." A vírgula marca a pausa natural antes da consequência.

2
SO + adjetivo/advérbio = tão / muito (intensidade)
SO é mais emocional que VERY — expressa sentimento
so + adj
She is so beautiful!
so + adj
This is so difficult.
so + adv
He runs so fast.
so much
I love you so much.
so many
So many people here!
so… that
So tired that she slept.

SO vs VERY: ambos = muito. Mas SO é mais emocional e espontâneo na fala: "It's SO good!" — ênfase real. VERY é mais neutro e escrito.

3
SO THAT = para que / a fim de que (propósito)
Indica intenção — diferente de SO (consequência)
TipoInglêsPortuguês
propósitoI study so that I can get a good job.Estudo para que possa ter um bom emprego.
propósitoSpeak slowly so that I understand.Fale devagar para que eu entenda.
propósitoI left early so that I wouldn't be late.Saí cedo para não me atrasar.
SO = resultado
It rained, so I stayed.
consequência — não planejado
SO THAT = propósito
I study so that I improve.
intenção — planejado
4
SO AM I — concordância positiva (eu também)
SO no início de resposta + SO como marcador de discurso
— I'm tired.
— So am I. (Eu também.)
— She can swim.
— So can he. (Ele também.)
— I love pizza.
— So do I. (Eu também.)
SO como marcador de discurso
So, what do you think? — Então, o que você acha?
So, let's get started. — Então, vamos começar.
So… I don't know how to say this. — Hesitação / início de assunto difícil.
Conectivo 2
BECAUSE — Porque / Por Causa De
1
BECAUSE = porque — explica a causa em qualquer tempo
Responde à pergunta WHY (por que) — sempre
TempoInglêsPortuguês
presenteI study English because I want a better job.Estudo inglês porque quero um emprego melhor.
presenteHe doesn't eat meat because he is vegetarian.Não come carne porque é vegetariano.
passadoI stayed home because it was raining.Fiquei em casa porque estava chovendo.
passadoShe cried because she was sad.Ela chorou porque estava triste.
futuroI will study more because I want to improve.Vou estudar mais porque quero melhorar.
negativaI don't go out because I'm tired.Não saio porque estou cansado.
2
BECAUSE no início da frase — ênfase na razão
Quando no início, use vírgula após a primeira oração
PosiçãoInglêsPortuguês
inícioBecause it was raining, I stayed home.Porque estava chovendo, fiquei em casa.
inícioBecause she studied hard, she passed.Porque ela estudou muito, ela passou.
meioI missed the bus because I was late.Perdi o ônibus porque me atrasei.
3
BECAUSE OF = por causa de (antes de substantivo)
BECAUSE + oração · BECAUSE OF + substantivo/pronome
BECAUSE + oração
I stayed home because it was raining.
porque + sujeito + verbo
She was late because the traffic was bad.
porque + sujeito + verbo
BECAUSE OF + substantivo
I stayed home because of the rain.
por causa de + substantivo
She was late because of the traffic.
por causa de + substantivo

BECAUSE + oração completa (sujeito + verbo) · BECAUSE OF + substantivo. Nunca: "because of it rained" — use "because it rained".

4
BECAUSE vs SINCE vs AS — todos = porque
Mesma ideia, registros diferentes — BECAUSE é sempre seguro
PalavraRegistroUsoExemplo
BECAUSEneutrorazão principalI left because I was tired.
SINCEformalrazão já conhecidaSince you're here, let's start.
ASformalrazão simultâneaAs it was raining, we stayed.

Na prática: use sempre BECAUSE — é neutro e funciona em qualquer contexto. SINCE e AS são mais formais e implicam que a razão já é conhecida pelo ouvinte.

"The reason is because..." — ERRADO. É redundante. Use: "The reason is that I was tired." ou simplesmente "I left because I was tired."
