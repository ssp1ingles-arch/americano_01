1
LEARN — quatro estruturas + americano vs britânico
LEARN + algo · + TO · + HOW TO · + ABOUT · + FROM — e a diferença LEARNED vs LEARNT
LEARN + algo
aprender algo
"I'm learning English."
LEARN + TO + BASE
aprender a fazer algo
"She learned to drive at 18."
LEARN + HOW TO
aprender como fazer algo
"I want to learn how to cook."
LEARN FROM
aprender com (erro / experiência)
"Learn from your mistakes."
🇺🇸
Inglês Americano
LEARNED / LEARNED
Passado e particípio com -ED — o mais comum no Brasil e no mundo digital
🇬🇧
Inglês Britânico
LEARNT / LEARNT
Forma irregular britânica — também válida, mas menos usada no contexto brasileiro
LEARN TO vs LEARN HOW TO — qual a diferença?
LEARN TO drive = aprender a dirigir (foco no resultado)
LEARN HOW TO drive = aprender como se dirige (foco no processo)
Na prática, são quase intercambiáveis. "I'm learning TO cook." / "I'm learning HOW TO cook." — ambas corretas e naturais.
LEARN é regular no inglês americano: LEARN → LEARNED. No inglês britânico, usa-se LEARNT. No Brasil, LEARNED é a forma ensinada e mais comum. As duas são aceitas em qualquer contexto.

⬤ Erros Clássicos dos Brasileiros
✗ "I didn't learned anything."
✓ "I didn't learn anything."
DIDN'T + BASE (learn). "I didn't LEARN anything." ✓ — DIDN'T já carrega o passado; verbo volta à base.
✗ "She learn languages quickly."
✓ "She learns languages quickly."
HE / SHE / IT → LEARNS (com -S). "She LEARNS languages quickly." ✓ — presente com SHE sempre com -S.
✗ "I want to learn how cooking."
✓ "I want to learn how to cook."
LEARN HOW TO + BASE. "I want to learn how TO COOK." ✓ — não -ING depois de HOW TO; base sempre.
✗ "Did you learned to drive?"
✓ "Did you learn to drive?"
DID + BASE. "Did you LEARN to drive?" ✓ — depois de DID, base LEARN (nunca LEARNED).
✗ "I'm learning to speak English since 2020."
✓ "I've been learning English since 2020."
Com SINCE (desde), use present perfect contínuo: "I've BEEN LEARNING English since 2020." — o -ING simples não combina com SINCE em inglês.
✗ "I will learned how to cook."
✓ "I will learn how to cook."
WILL + BASE. "I will LEARN how to cook." ✓ — base sempre depois de modais.
◈ Quiz — LEARN
Pergunta 1 de 12
Qual é o passado correto de LEARN no inglês americano?
No inglês americano (padrão no Brasil): LEARN → LEARNED. "Learnt" é a forma britânica — também correta, mas menos usada no contexto brasileiro.
Pergunta 2 de 12
"Ela aprende idiomas rapidamente." → Presente com SHE.
HE / SHE / IT → LEARNS (com -S). "She LEARNS languages quickly." ✓ — hábito/característica presente com SHE.
Pergunta 3 de 12
"Eu quero aprender a cozinhar." → LEARN + TO.
LEARN + TO + BASE. "I want to learn TO COOK." ✓ — também correto: "learn HOW TO cook." Nunca "learn cooking" (sem TO) nem "how cooking" (sem TO).
Pergunta 4 de 12
"Aprendi a dirigir quando tinha 18 anos." → Passado.
Passado afirmativo regular: LEARNED direto (sem auxiliar). "I LEARNED to drive when I was 18." ✓
Pergunta 5 de 12
"Você aprendeu muito neste curso?" → Passado, pergunta.
DID + BASE. "Did you LEARN a lot in this course?" ✓ — depois de DID, base LEARN (nunca LEARNED).
Pergunta 6 de 12
"Aprenda com seus erros." → Imperativo.
Imperativo = base do verbo diretamente. "LEARN from your mistakes." ✓ — sem auxiliar, sem -ING, sem -ED.
Pergunta 7 de 12
"Ela não aprendeu nada." → Passado, negativo.
DIDN'T + BASE. "She didn't LEARN anything." ✓ — DIDN'T já carrega o passado; verbo vai à base LEARN.
Pergunta 8 de 12
LEARN TO vs LEARN HOW TO — qual é a diferença prática?
"Learn TO cook" e "Learn HOW TO cook" são ambas naturais e corretas. HOW TO dá um leve foco em como se faz o processo; TO foca no resultado. Na prática do dia a dia, são intercambiáveis.
Pergunta 9 de 12
"Eu vou aprender a programar." → WILL.
WILL + BASE. "I will LEARN how to code." ✓ — base sempre depois de modais.
Pergunta 10 de 12
"Estou aprendendo inglês." → Contínuo presente.
"I'm learning English." = presente contínuo, ação em andamento. ✓ — LEARN aceita perfeitamente o -ING (diferente de UNDERSTAND/KNOW que são verbos de estado).
Pergunta 11 de 12
"Ele não aprende com os erros." → Presente com HE, negativo.
HE → DOESN'T (não DON'T). Depois de DOESN'T: BASE sem -S. "He DOESN'T LEARN from mistakes." ✓
Pergunta 12 de 12
"Vamos aprender juntos." → GOING TO.
WE → ARE going to + BASE. "We ARE going to LEARN together." ✓ — base LEARN (não LEARNED) depois de GOING TO.
