Por que o inglês "gruda" as palavras

"Did you" vira "didja", "want to" vira "wanna". Não é preguiça — são 4 fenômenos com regras claras. Entenda cada um e o inglês falado deixa de ser um borrão sonoro.

🔗
Fenômeno 1
Linking — Ligação de Sons
O que é: quando uma palavra termina em consoante e a próxima começa em vogal, os dois sons se juntam e as palavras soam como uma só.
Exemplos
EscritoFaladoExplicação
turn it offtur-ni-toffturn + it + off soam como uma palavra só
pick it uppi-ki-tuppick + it + up se encadeiam
get outge-TAUTo T de "get" cola no AU de "out"
come onka-MONcome + on = "c'mon"
hold onhol-DONo D de "hold" vai pro "on"
💡 Dica para o brasileiro
Em português a gente faz igual! "Faz assim" vira "fa-ZA-sim". Em inglês é o mesmo princípio, só que mais rápido.
Reconheça o linking
Como as palavras se encadeiam?
Como soa turn it off?
tur-ni-toffturn · it · off (separado)tur-nof
Como soa get out?
get · out (separado)ge-tautgout

Fenômeno 2
Elision — Sons que Desaparecem
O que é: alguns sons simplesmente são engolidos na fala rápida. Você espera ouvi-los, mas eles somem.
Exemplos
EscritoFaladoO que sumiu
next daynex dayo T de "next" desaparece
last nightlas nighto T de "last" desaparece
must bemuss beo T de "must" desaparece
and'n'vira só o N entre palavras
him'imo H inicial some
her'ero H inicial some
them'emo TH inicial some
📏 Regra
Os sons que mais desaparecem: T antes de consoante (next day, last night), H em palavras átonas (him, her, his) e TH em "them".
Reconheça a elisão
Qual som desaparece?
Em last night, o que some?
o T de "last"o N de "night"nada some
Como se fala tell them rápido?
tell demtell 'emtell zem

Fenômeno 3
Assimilation — Sons que se Transformam
O que é: um som muda de forma para facilitar a articulação com o som vizinho. É isso que explica a maioria das reduções mais confusas.
Exemplos
EscritoFaladoO que aconteceu
did youdidjaD + Y = J (como em "dj")
would youwouldjaD + Y = J
could youcouldjaD + Y = J
what youwhatchaT + Y = CH (como "tch")
got yougotchaT + Y = CH
meet youmeetchaT + Y = CH
don't youdonchaT + Y = CH
can't youcanchaT + Y = CH
⭐ Regra de ouro
D + Y = "dj" (como em "jeans"). T + Y = "tch" (como em "tchau"). Isso sozinho explica metade das reduções mais confusas!
Reconheça a assimilação
Que som se forma?
did you vira...
didja (som de "dj")dicha (som de "tch")did-you (igual)
got you vira...
gotja (som de "dj")gotcha (som de "tch")got-you (igual)

Fenômeno 4
Weak Forms — Palavras que Encolhem
O que é: palavras pequenas (preposições, artigos, auxiliares) têm uma forma fraca quando não são enfatizadas. É a forma fraca que você ouve 90% do tempo.
Exemplos
PalavraForte → FracaExemplo
and/ænd/ → /ən/, /n/"fish 'n' chips"
to/tuː/ → /tə/"going to" → "gonna"
for/fɔːr/ → /fər/"I did it fer you"
of/ɒv/ → /əv/, /ə/"lotta" = lot of
have/hæv/ → /həv/, /əv/"shoulda" = should have
the/ðiː/ → /ðə/"gimme the ball"
a/eɪ/ → /ə/"give me a sec"
them/ðem/ → /əm/"tell 'em"
him/hɪm/ → /ɪm/"ask 'im"
her/hɜːr/ → /ər/"call 'er"
🎯 Por que isso importa
Em inglês falado rápido, 90% das palavras funcionais usam a forma fraca. Se você espera ouvir "and" como /ænd/, vai perder metade das frases.
Reconheça a forma fraca
Qual é a forma fraca?
A forma fraca de and em "fish and chips" é...
'n' (som de N)ãnd (bem forte)end
"should have" reduz para...
should-hav (forte)shoulda (have vira "a")should-of
