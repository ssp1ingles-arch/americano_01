1
THINK — mnemônico e família -OUGHT
THINK → THOUGHT — família dos verbos que terminam em -OUGHT / -AUGHT
Mnemônico — família -OUGHT / -AUGHT
THINK → THOUGHT — a raiz muda completamente. Lembre a família que termina em -OUGHT ou -AUGHT no passado e particípio:
THINK → THOUGHT BRING → BROUGHT BUY → BOUGHT FIGHT → FOUGHT SEEK → SOUGHT CATCH → CAUGHT TEACH → TAUGHT
Uso mais frequente — opinião e crença
I THINK (THAT) + sujeito + verbo = acho que / acredito que
"I think (that) you're right." = Acho que você tem razão.
"She thinks (that) it's a good idea." = Ela acha que é uma boa ideia.
"I don't think (that) it will work." = Não acho que vai funcionar.
THAT é opcional — pode ser omitido na fala informal.
THINK (THAT)
opinião / crença
"I think it's great."
"She thinks he's wrong."
→ o que você acha?
THINK ABOUT
refletir / considerar
"Think about it!"
"I'm thinking about changing jobs."
→ processo de reflexão
THINK OF
opinião / lembrar / criar
"What do you think of it?"
"I can't think of his name."
→ opinião ou memória
THINK ABOUT vs THINK OF: em muitos casos são intercambiáveis. THINK ABOUT = processo de reflexão (mais longo). THINK OF = opinião imediata ou lembrar algo. "What do you think OF the movie?" (opinião) / "I've been thinking ABOUT the movie all day." (reflexão contínua).

4
I THINK SO · I DON'T THINK SO
Formas curtas para concordar ou discordar com suavidade
Concordar / presumir positivo
I think so.Acho que sim.
I believe so.Acredito que sim.
I suppose so.Suponho que sim.
I hope so.Espero que sim.
Discordar / presumir negativo
I don't think so.Acho que não.
I don't believe so.Não acredito que sim.
I don't suppose so.Não suponho que sim.
I hope not.Espero que não.
Atenção: "I think NOT." ✗ (muito formal/arcaico) / "I DON'T think so." ✓ — a negação fica em THINK, não em SO. "I think SO not." ✗ — inválido. A ordem é: I + DON'T THINK + SO.

⬤ Erros Clássicos dos Brasileiros
✗ "I thinked it was a good idea."
✓ "I thought it was a good idea."
THINK é irregular: THINK → THOUGHT. "I THOUGHT it was a good idea." ✓ — "thinked" não existe. Família -OUGHT: THINK → THOUGHT / BRING → BROUGHT / BUY → BOUGHT.
✗ "I'm thinking you're right." (opinião)
✓ "I think you're right."
Para opiniões, THINK não usa contínuo. "I THINK you're right." ✓ — THINK de crença é estativo. O contínuo THINKING é válido somente para processo ativo: "I'm thinking about the problem." ✓ (estou refletindo).
✗ "I think not." (resposta negativa informal)
✓ "I don't think so."
"I DON'T THINK so." ✓ — a negação fica em THINK. "I think NOT" é arcaico/literário. Compare: "I hope NOT." ✓ — HOPE é exceção onde NOT fica no final.
✗ "What did you thought of the movie?"
✓ "What did you think of the movie?"
DID + BASE. "What did you THINK of the movie?" ✓ — depois de DID, base THINK (nunca THOUGHT).
✗ "I'm thinking to move to another city."
✓ "I'm thinking of/about moving to another city."
THINK OF/ABOUT + -ING (não TO + base). "I'm thinking OF MOVING to another city." ✓ / "I'm thinking ABOUT MOVING." ✓ — para planos futuros. Compare: "I'm planning TO MOVE." ✓ — PLAN usa TO.
✗ "She think he's wrong."
✓ "She thinks he's wrong."
SHE → THINKS (com -S). "She THINKS he's wrong." ✓ — terceira pessoa singular sempre com -S no presente simples.
◈ Quiz — THINK
Pergunta 1 de 12
Qual é o passado de THINK?
THINK → THOUGHT (irregular). Família -OUGHT: THINK → THOUGHT / BRING → BROUGHT / BUY → BOUGHT. "thinked" não existe.
Pergunta 2 de 12
"Acho que você tem razão." → Opinião com I THINK.
"I THINK you're right." ✓ — para opiniões, THINK no presente simples. "I'm thinking" ✗ para opinião — contínuo é para processo ativo. "I thought you're right" ✗ — mistura passado com presente.
Pergunta 3 de 12
"O que você achou do filme?" → THINK OF, passado.
"What did you THINK of the movie?" ✓ — DID + base THINK. "Did you thought" ✗ — depois de DID, base (nunca THOUGHT). Sem DID no início, não forma pergunta correta.
Pergunta 4 de 12
"Estou pensando em mudar de emprego." → THINK ABOUT + -ING.
"I'm thinking about CHANGING jobs." ✓ — THINK ABOUT + -ING para planos futuros. "Thinking TO change" ✗ — TO é inválido após THINK ABOUT/OF. "I think about to change" ✗ — não existe essa estrutura.
Pergunta 5 de 12
"Acho que não." → Resposta negativa curta com THINK.
"I DON'T THINK so." ✓ — a negação fica em THINK. "I think NOT" é arcaico/literário. "I think SO not" ✗ — não existe. Contraste: "I HOPE not." ✓ — HOPE é exceção com NOT no final.
Pergunta 6 de 12
"Nunca achamos que terminaria assim." → Passado com WE + NEVER.
"We never THOUGHT it would end this way." ✓ — THOUGHT = passado de THINK. "thinked" ✗. "we never think" = presente (hábito), não passado.
Pergunta 7 de 12
Qual é a diferença entre THINK ABOUT e THINK OF?
THINK ABOUT = processo de reflexão (mais longo). THINK OF = opinião imediata ou lembrar algo. "What do you think OF it?" (opinião instantânea). "I've been thinking ABOUT the offer all week." (reflexão prolongada). Em muitos contextos são intercambiáveis.
Pergunta 8 de 12
"Ela acha que é uma ótima oportunidade." → HE/SHE.
SHE → THINKS (com -S). "She THINKS it's a great opportunity." ✓ — terceira pessoa com -S. "She think" ✗. "She is thinking it's a great opportunity" ✗ — THINK de opinião não usa contínuo.
Pergunta 9 de 12
O que significa "Come to think of it, you're right!"?
"Come to THINK of it" = agora que penso nisso / aliás — expressão que introduz algo que acabou de ser lembrado ou percebido. Usado quando alguém reconsideira e concorda com algo.
Pergunta 10 de 12
"I had ___ about it before." → Past perfect, particípio.
"I had THOUGHT about it before." ✓ — HAD + particípio THOUGHT. Past perfect = ação anterior a outra no passado. Particípio de THINK = THOUGHT (igual ao passado).
Pergunta 11 de 12
"Pense bem e me avise." → THINK IT OVER.
"THINK IT OVER and let me know." ✓ — phrasal verb separável: THINK OVER = refletir/ponderar. "Think it TWICE" ✗ — a expressão correta é THINK TWICE (pensar duas vezes, sem IT). "Think OVER it" ✓ também é possível com pronome.
Pergunta 12 de 12
Qual verbo da família -OUGHT/AUGHT NÃO pertence ao grupo?
FEEL → FELT não pertence à família -OUGHT. FEEL é da família EE → E + T (como KEEP → KEPT). A família -OUGHT: THINK → THOUGHT / BRING → BROUGHT / BUY → BOUGHT / FIGHT → FOUGHT / SEEK → SOUGHT.
