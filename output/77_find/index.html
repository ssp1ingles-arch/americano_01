1
FIND — mnemônico e padrões essenciais
FIND → FOUND — família -IND → -OUND · FIND vs FIND OUT
Mnemônico — família -IND → -OUND
FIND → FOUND — a vogal -IND muda para -OUND. Lembre a família:
FIND → FOUND BIND → BOUND WIND → WOUND GRIND → GROUND
FIND · encontrar
FIND
objeto físico ou pessoa
"I found my phone!"
(encontrei o celular — físico)

"Did you find a solution?"
(encontrou uma solução?)
FIND OUT · descobrir
FIND OUT
informação · fato · verdade
"I found out the truth."
(descobri a verdade — informação)

"She found out he was lying."
(ela descobriu que ele mentia)
Padrão especial — FIND + IT + adjetivo (+ TO + base)
FIND IT + adj. = achar algo [difícil/fácil/interessante]
"I find it interesting." = Acho interessante.
"I find it hard to wake up early." = Acho difícil acordar cedo.
"She found it easy to learn." = Ela achou fácil de aprender.
IT é obrigatório nessa estrutura — não omita: "I find hard" ✗ → "I find IT hard" ✓
FIND + objeto
encontrar (físico/concreto)
"I can't find my wallet." / "Did you find a job?"
FIND + obj. + ADJ
achar / considerar (qualidade)
"I found the test difficult." / "She finds him boring."
FIND OUT
descobrir (informação/fato)
"Find out what time it starts." / "I found out he lied."
FIND YOURSELF + -ING
se ver fazendo algo (sem perceber)
"I found myself laughing." / "She found herself alone."

⬤ Erros Clássicos dos Brasileiros
✗ "I finded my keys."
✓ "I found my keys."
FIND é irregular: FIND → FOUND. "I FOUND my keys." ✓ — "finded" não existe. Família -IND → -OUND: FIND → FOUND / BIND → BOUND / WIND → WOUND.
✗ "I find hard to wake up early."
✓ "I find it hard to wake up early."
FIND IT + adjetivo — IT é obrigatório. "I find IT hard to wake up early." ✓ — IT é o objeto formal da frase. "I find hard" ✗ — FIND precisa de objeto direto.
✗ "I didn't found the answer."
✓ "I didn't find the answer."
DIDN'T + base. "I didn't FIND the answer." ✓ — depois de DIDN'T, base FIND (nunca FOUND).
✗ "She found the truth." (queria dizer: descobriu a verdade)
✓ "She found OUT the truth."
Para descobrir informação, use FIND OUT. "She found OUT the truth." ✓ / "She found out THAT he lied." ✓ — FIND (sem OUT) = encontrar objeto físico ou considerar. "She found the truth" pode significar que ela fisicamente encontrou algo chamado "verdade".
✗ "Find out it before tomorrow." (com pronome)
✓ "Find it out before tomorrow."
Com pronome, separação é obrigatória: "Find IT out." ✓ / "Find out IT." ✗ — pronomes sempre ficam no meio do phrasal verb separável. Com substantivos: "Find OUT the answer." ✓ ou "Find the answer OUT." ✓
✗ "She find it interesting."
✓ "She finds it interesting."
SHE → FINDS (com -S). "She FINDS it interesting." ✓ — terceira pessoa singular sempre com -S no presente simples.
◈ Quiz — FIND
Pergunta 1 de 12
Qual é o passado de FIND?
FIND → FOUND (irregular). Família -IND → -OUND: FIND → FOUND / BIND → BOUND. "finded" não existe. Passado = particípio = FOUND.
Pergunta 2 de 12
Qual é a diferença entre FIND e FIND OUT?
FIND = encontrar fisicamente. FIND OUT = descobrir informação. "I FOUND my phone." (físico). "I FOUND OUT what happened." (informação). Para descobrir a verdade, segredo ou fato, use FIND OUT.
Pergunta 3 de 12
"Acho difícil acordar cedo." → FIND IT + adjetivo.
"I find IT hard to wake up early." ✓ — IT é obrigatório. Estrutura: FIND + IT + adjetivo + TO + base. "I find hard" ✗ — FIND precisa de objeto (IT). "Find that hard" ✗ — estrutura inválida.
Pergunta 4 de 12
"Ela descobriu que sua amiga havia se mudado." → FIND OUT.
"She FOUND OUT that her friend had moved." ✓ — FIND OUT para descobrir informação. "She found that" ✓ é gramaticalmente aceitável, mas "found out that" é mais preciso para descoberta. "finded out" ✗ — passado de FIND = FOUND.
Pergunta 5 de 12
"Não consigo encontrar minha carteira em lugar nenhum."
"I can't FIND my wallet anywhere." ✓ — CAN'T + base FIND. "Can't found" ✗ — depois de modal, base FIND. "I don't find anywhere" ✗ — para busca ativa em progresso, CAN'T FIND é muito mais natural que DON'T FIND.
Pergunta 6 de 12
"Você encontrou o que estava procurando?" → Passado, pergunta.
"Did you FIND what you were looking for?" ✓ — DID + base FIND. "Did you FOUND" ✗ — depois de DID, base (nunca FOUND). "Have you FOUND" ✓ também é correto (present perfect), com nuance diferente.
Pergunta 7 de 12
"Ela se viu chorando sem perceber." → FIND YOURSELF + -ING.
"She found herself CRYING." ✓ — FIND YOURSELF + -ING = se ver fazendo algo (sem perceber). "Found herself to cry" ✗ — TO é inválido nessa estrutura. "To crying" ✗ — forma híbrida inválida.
Pergunta 8 de 12
Com pronome, qual é a posição correta em FIND OUT?
"Find IT out before tomorrow." ✓ — com pronome, separação é obrigatória: pronome no meio. "Find out it" ✗ — pronomes não vêm depois da partícula. Com substantivos: "Find OUT the answer." ✓ ou "Find the answer OUT." ✓
Pergunta 9 de 12
"Achamos a reunião muito produtiva." → WE + FIND + obj. + adj., passado.
"We found the meeting very PRODUCTIVE." ✓ — FIND + objeto + adjetivo (não advérbio). "Productively" ✗ — é advérbio. "We found IT productive" também seria correto com IT como objeto formal.
Pergunta 10 de 12
"I have ___ a great solution!" → Present perfect.
"I have FOUND a great solution!" ✓ — HAVE + particípio FOUND. "I have find" ✗ — depois de HAVE, particípio (não base). "finded" ✗ — não existe.
Pergunta 11 de 12
O que significa "He always finds fault with my work"?
FIND FAULT WITH = encontrar defeitos em / criticar. "He always FINDS FAULT with my work." = ele sempre arranja uma crítica ao meu trabalho. É uma expressão idiomática com conotação negativa.
Pergunta 12 de 12
Qual verbo pertence à mesma família irregular de FIND → FOUND?
BIND → BOUND pertence à família -IND → -OUND, igual a FIND → FOUND. THINK → THOUGHT é família -OUGHT. KEEP → KEPT é família EE → E + T. As famílias ajudam a memorizar grupos de irregulares juntos.
