"We need to talk."

1
TALK — quatro formas de usar
TALK TO · TALK ABOUT · TALK OVER · TALK someone INTO
TALK TO
falar com / conversar com
"I talk to my mom every day."
TALK ABOUT
falar sobre / discutir
"He talks about soccer all the time."
TALK IT OVER
resolver conversando
"Let's talk it over."
TALK someone INTO
convencer alguém a
"She talked me into going."
TALK = sempre informal e sobre conversação. SPEAK = formal ou idiomas. "We need to TALK." = frase clássica do inglês — nativo usa TALK, não SPEAK.

TALK é REGULAR → TALKED. Diferente de SPEAK → SPOKE (irregular). Na negação volta à base: "I didn't TALK" ✓ | "I didn't talked" ✗
4
Perguntas e Negação
Mesma regra: DO / DOES / DID → verbo volta à BASE
Como fazer perguntas com TALK
Presente
DO you / we / they + talk?  |  DOES he / she / it + talk?
"Do you talk to her?" / "Does he talk about soccer?"
Passado
DID + sujeito + talk?
"Did you talk to him?" / "Did they talk about everything?"
DO/DOES/DID vão para o início → verbo volta à FORMA BASE. "Did he TALK?" ✓ | "Did he TALKED?" ✗
Negação com TALK
Presente
I/YOU/WE/THEYdon't talknão converso/a
HE/SHE/ITdoesn't talknão conversa
Passado / Futuro
todosdidn't talknão conversou
todoswon't talknão vai conversar
"I didn't talked to her." ✗ → "I didn't talk to her." ✓ — DIDN'T já marca passado. Verb = base.
5

⬤ Erros Clássicos dos Brasileiros
✗ "I didn't talked to her."
✓ "I didn't talk to her."
Com DIDN'T o verbo volta à BASE (talk). DIDN'T já marca o passado — nunca adicione -ED depois.
✗ "She talk too much."
✓ "She talks too much."
HE / SHE / IT no presente = sempre TALKS (com -S). "The parrot TALKS!" ✓
✗ "Do you talk English?"
✓ "Do you speak English?"
Idioma = sempre SPEAK. TALK = conversar (sobre assuntos), não falar um idioma.
✗ "He doesn't talks about it."
✓ "He doesn't talk about it."
DOESN'T já marca 3ª pessoa — o verbo principal vai para a BASE, sem -S.
✗ "We need to speak." (pedir uma conversa informal)
✓ "We need to talk."
Para pedir uma conversa casual/emocional o nativo usa TALK. SPEAK soa formal demais.
✗ "I will to talk to her tomorrow."
✓ "I will talk to her tomorrow."
Depois de WILL nunca use TO. WILL + forma base diretamente.

Pergunta 9 de 12
