1
START + -ING ou START + TO — qual usar?
As duas estruturas existem — mas START + -ING é mais natural na fala cotidiana
✓ Mais comum (fala)
START + -ING
"She started crying." ✓
"It started raining." ✓
"He started working here." ✓
✓ Também correto (mais formal)
START + TO + base
"She started to cry." ✓
"It started to rain." ✓
"He started to work here." ✓
Ambas as formas estão corretas! START + -ING é mais frequente no inglês cotidiano. START TO + base aparece mais em contextos formais e escritos. O significado é idêntico.
Expressão-chave — GET STARTED
LET'S GET STARTED / GET STARTED!
Vamos começar! / Mãos à obra! / Vamos lá!
"Let's get started — we don't have much time." · "Are you ready to get started?" · "I'll get started right away."
Quatro usos principais de START
UsoExemploTradução
START + substantivoStart the car.Ligue o carro.
START + -INGShe started crying.Ela começou a chorar.
START A + negócioHe started a company.Ele fundou / abriu uma empresa.
START FROM SCRATCHWe had to start from scratch.Tivemos que começar do zero.

Futuro e Present Perfect
HAVE STARTED + HAVE BEEN + -ING — "I've just started" é muito frequente
Futuro

ITThe movie is going to start soon.O filme vai começar em breve.
WEWe won't start without you.Não vamos começar sem você.
Present Perfect

"I've just started." = Acabei de começar. · "Have you started yet?" = Você já começou? · "We haven't started yet." = Ainda não começamos. Expressões clássicas do cotidiano!

⬤ Erros Clássicos dos Brasileiros
✗ "She start crying."
✓ "She started crying."
Passado → STARTED (regular). "She STARTED crying." ✓ — base + ED. Também aceito: "She started TO CRY." ✓
✗ "The movie start at 9."
✓ "The movie starts at 9."
THE MOVIE (IT) → STARTS (com -S). "The movie STARTS at 9." ✓ — terceira pessoa singular sempre com -S.
✗ "I didn't started the project yet."
✓ "I haven't started the project yet."
Com YET → present perfect. "I HAVEN'T STARTED the project yet." ✓ — YET pede HAVE + particípio, não DIDN'T + base.
✗ "Let's start to learning."
✓ "Let's start learning." / "Let's start to learn."
START + -ING = start learning ✓ · START + TO + base = start to learn ✓ · START + TO + -ING = "start to learning" ✗ — mistura inválida.
✗ "We started the class but didn't started the exam."
✓ "We started the class but didn't start the exam."
DIDN'T + BASE. "We didn't START the exam." ✓ — depois de DIDN'T, base START (nunca STARTED).
✗ "It started to raining."
✓ "It started raining." / "It started to rain."
Escolha um: START + -ING = "started raining" ✓ · START + TO + base = "started to rain" ✓ · "started to raining" mistura as duas formas — inválido.
◈ Quiz — START
Pergunta 1 de 12
Qual é o passado de START?
START → STARTED (regular). Base + ED. "She STARTED crying." ✓
Pergunta 2 de 12
"O filme começa às 21h." → Hábito com IT.
THE MOVIE (IT) → STARTS (com -S). "The movie STARTS at 9 p.m." ✓ — hábito/horário fixo com simples presente.
Pergunta 3 de 12
"Ela começou a chorar ao ouvir a notícia." → START + ação.
"She STARTED CRYING when she heard the news." ✓ — START + -ING. Também correto: "She started TO CRY." ✓ — mas "started to crying" ✗ mistura as duas formas.
Pergunta 4 de 12
"Vamos começar! / Mãos à obra!" → Expressão com GET.
"Let's GET STARTED!" ✓ — expressão fixa muito usada para iniciar algo. GET STARTED = começar / dar o pontapé inicial.
Pergunta 5 de 12
"Ainda não começamos." → Present perfect com YET.
"We HAVEN'T STARTED yet." ✓ — YET + present perfect negativo. "We didn't start yet." ✗ (YET pede present perfect em inglês britânico/formal; no americano informal é aceito, mas HAVEN'T STARTED YET é o padrão).
Pergunta 6 de 12
"Quando você começou a trabalhar aqui?" → Passado, pergunta.
DID + BASE. "When did you START working here?" ✓ — depois de DID, base START (nunca STARTED).
Pergunta 7 de 12
O que significa "start from scratch"?
"Start from scratch" = começar do zero. "We had to start from scratch after the fire destroyed everything." ✓ — expressão idiomática, SCRATCH = marcação de linha inicial em corridas antigas.
Pergunta 8 de 12
"Não vamos começar sem você." → Futuro negativo.
"We WON'T START without you." ✓ — futuro negativo: WON'T + base START. "We don't start without you." = presente habitual (regra geral). "We didn't start" = passado.
Pergunta 9 de 12
"Ela abriu seu próprio negócio ano passado." → START + substantivo.
"She STARTED HER OWN BUSINESS last year." ✓ — START + substantivo (negócio). "She started UP a business." ✓ também (phrasal verb START UP = fundar).
Pergunta 10 de 12
"Acabei de começar." → Present perfect com JUST.
"I'VE JUST STARTED." ✓ — present perfect com JUST para ação recém-concluída. "I just started." ✓ também (inglês americano informal aceita o simples passado com JUST).
Pergunta 11 de 12
Qual estrutura é INVÁLIDA com START?
"She started to CRYING." ✗ — mistura inválida das duas formas. Escolha uma: START + -ING ("started crying" ✓) OU START + TO + base ("started to cry" ✓).
Pergunta 12 de 12
O que é um "head start"?
"A head start" = vantagem inicial. "Learning English early gives you a head start in your career." — expressão de esporte (corrida) usada em contextos gerais para indicar vantagem competitiva.
