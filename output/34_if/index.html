Dois tipos de IF — real e hipotético
IF — os dois usos principais
IF real — Esta lição
IF + presente → WILL
If it rains, I'll stay home.
Se chover (possível), ficarei em casa.
situação possível — pode acontecer
IF hipotético — Lição 31
IF + passado → WOULD
If I had money, I would travel.
Se eu tivesse dinheiro (irreal), viajaria.

Uso 1
IF Real — Situação Possível
1
IF + presente simples → WILL + verbo base
Condição possível no presente ou futuro — pode acontecer
IF + presente simples , WILL + verbo base
EstruturaInglêsPortuguês
if + pres/willIf it rains, I'll stay home.Se chover, fico em casa.
if + pres/willIf you study, you'll pass.Se você estudar, você vai passar.
if + pres/willIf she calls, tell her I'm busy.Se ela ligar, diz que estou ocupado.

⚠ WILL nunca vai depois de IF condicional

✗ errado
If it will rain, I'll stay home.
✓ correto
If it rains, I'll stay home.
✗ errado
If she will call, tell her I'm busy.
✓ correto
If she calls, tell her I'm busy.
REGRA: depois de IF condicional use o PRESENTE SIMPLES (rains, calls, is, go) — nunca WILL. O WILL fica na outra parte da frase.

A ordem pode ser invertida: "If it rains, I'll stay" = "I'll stay if it rains." Quando IF começa a frase, use vírgula depois da condição. Quando IF está no meio, sem vírgula.

Uso 2
IF em Perguntas Indiretas = WHETHER
2
IF = WHETHER — "se" em pergunta indireta
Não é condicional — expressa dúvida ou incerteza
IF/WHETHER
I don't know if she's coming.
Não sei se ela vem.
IF/WHETHER
Can you check if the door is locked?
Você pode verificar se a porta está trancada?
IF/WHETHER
Tell me if you need help.
Me diga se você precisar de ajuda.
IF/WHETHER
I wonder if he knows.
Fico me perguntando se ele sabe.
Pergunta direta
"Is she coming?"
pergunta direta — inversão
Pergunta indireta (IF)
"I don't know if she's coming."
sem inversão — ordem direta

IF = WHETHER em perguntas indiretas. Os dois são corretos: "I don't know if/whether she's coming." Na pergunta indireta, use ordem direta — sem inversão: "I don't know if she IS coming" (não "if IS she coming").

Uso 3
IF no Dia a Dia — Polidez, Ofertas e Condições
3
IF em pedidos educados, ofertas e condições especiais
Um dos conectivos mais versáteis do inglês falado
Polidez
I'd be grateful if you could help.
Ficaria grato se você pudesse ajudar.
Polidez
If you don't mind, I'll sit here.
Se não se importar, vou sentar aqui.
Oferta
If you want, I can help.
Se você quiser, posso ajudar.
Condição
Only if you promise.
Só se você prometer.
Surpresa
If that's true, I'm shocked.
Se isso for verdade, estou chocado.
Resposta WHY
Even if it rains, I'll go.
Mesmo se chover, irei.

IF é um dos conectivos mais usados em inglês — pedidos educados, condições, ofertas, perguntas indiretas. Dominar IF significa soar mais natural em qualquer conversa.
