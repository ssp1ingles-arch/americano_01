🧸 Infância & Used To
Used To — Como falar do que você COSTUMAVA fazer
Estrutura
TipoFórmulaExemplo
Afirmativosubject + used to + base verbI used to eat cereal for dinner.
Negativosubject + didn't use to + base verbI didn't use to like vegetables.
PerguntaDid + subject + use to + base verb?Did you use to play outside?
⚠️ CUIDADO: used to ≠ get used to
"I used to play outside"
Hábito do passado — algo que eu fazia regularmente e não faço mais.
"I got used to the noise"
Me acostumei com — passei a aceitar/tolerar algo que antes incomodava.
São expressões completamente diferentes — não confunda!
