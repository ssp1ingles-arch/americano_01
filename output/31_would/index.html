WOULD
Gostaria · Costumava · Seria — o modal mais versátil do inglês
I'd like = gostaria would = costumava If... would = seria would rather = preferiria
role para baixo
Uso 1
WOULD LIKE — Pedido Educado e Oferta
1
WOULD LIKE = gostaria — forma mais educada de querer algo
"Would you like...?" é a oferta mais educada em inglês
Pedido
Would you help me, please?
Você poderia me ajudar?
Desejo
I would like a coffee.
Eu gostaria de um café.
Oferta
Would you like some water?
Você gostaria de água?
Preferência
I would prefer to stay.
Eu preferiria ficar.
TipoInglêsPortuguês
I'd like I'd like to visit Paris. Eu gostaria de visitar Paris. (I'd = I would)
She'd like She'd like some tea. Ela gostaria de chá.
Would you? Would you like anything else? Você gostaria de mais alguma coisa?
I'd love I would love to help. Eu adoraria ajudar.

I'd = I would (contração). "I would like" = quer dizer "quero" de forma educada. Em restaurante, loja ou reunião, sempre prefira "I'd like" ao invés de "I want".

Uso 2 — Passado
WOULD — Hábito no Passado
2
WOULD + verbo base = costumava — ação repetida no passado
Equivale a USED TO para hábitos — não para estados
TipoInglêsPortuguês
hábito When I was young, I would play soccer every day. Quando jovem, eu costumava jogar futebol todo dia.
hábito She would always bring flowers. Ela sempre trazia flores (costumava trazer).
hábito We would go to the beach every summer. A gente costumava ir à praia todo verão.
WOULD de hábito = USED TO — "I would play" = "I used to play". Mas atenção: para estados (morar, ser, ter), só USED TO funciona:
✓ "I used to live there"  ·  ✗ "I would live there" (estado — não ação)
Uso 3 — Hipótese
WOULD — Condicional (Se... Então...)
3
IF + passado simples → WOULD + verbo base
Situação hipotética — não real agora
IF + passado simples , WOULD + verbo base
TipoInglêsPortuguês
cond. If I had money, I would travel. Se eu tivesse dinheiro, viajaria.
cond. She would come if she could. Ela viria se pudesse.
cond. What would you do? O que você faria?
cond. I would love to visit Japan. Eu adoraria visitar o Japão.

⚠ Erro clássico — WOULD não vai depois de IF

✗ errado
If I would have money, I would travel.
✓ correto
If I had money, I would travel.
REGRA: depois de IF, use o passado simples (had, were, could) — nunca WOULD. O WOULD fica na outra parte da frase: "If I had..., I would..."
Uso 4 — Preferência
WOULD RATHER · WOULD PREFER
4
WOULD RATHER / WOULD PREFER = preferiria
Estruturas diferentes — atenção ao uso de TO

WOULD RATHER

WOULD RATHER + verbo base
I would rather walk.Eu preferiria caminhar.
She would rather stay home.Ela preferiria ficar em casa.
Would you rather have tea?Você prefere chá?

WOULD PREFER

WOULD PREFER + to + infinitivo
I would prefer to eat now.Eu prefiro comer agora.
She'd prefer to stay.Ela preferiria ficar.
Would you prefer to walk?Você preferiria caminhar?

Diferença: WOULD RATHER + verbo base (sem TO) · WOULD PREFER + TO + infinitivo. "I'd rather walk" ✓ · "I'd rather to walk" ✗ · "I'd prefer to walk" ✓
