1
TRY — dois usos + dois phrasal verbs
TRY TO · TRY -ING · TRY ON · TRY OUT — cada um com significado diferente!
TRY TO + BASE
tentar fazer algo (esforço)
"I'm trying to sleep."
TRY + ING
experimentar fazer algo (ver o que acontece)
"Try eating less sugar."
TRY ON
experimentar (roupa, calçado)
"Can I try this on?"
TRY OUT
testar / experimentar algo novo
"Let's try out the new app."
TRY TO + BASE
TRY TO DO
= tentar fazer (esforço, nem sempre com sucesso)
"I tried to open the door."
Tentei abrir a porta (mas talvez não consegui).
TRY + DOING
TRY DOING
= experimentar fazer (ver se funciona / resolve)
"Try opening the window."
Experimenta abrir a janela (para ver se ajuda).
TRY ON
experimentar (roupa / sapato)
"Can I try this dress on?"
Posso experimentar este vestido?
TRY OUT
testar / dar uma chance a algo novo
"Let's try out this restaurant."
Vamos testar esse restaurante.
Atenção à ortografia — Y → IED
TRY → TRIED · TRIES (HE/SHE/IT) · TRYING (-ING normal)
Quando o verbo termina em consoante + Y: troca o Y por IED no passado e IES no presente HE/SHE/IT.
TRY é regular, mas tem mudança de grafia: TRY → TRIED (não "tryed"!). Regra: consoante antes do Y → troca Y por IED. Mesma regra de STUDY → STUDIED, CARRY → CARRIED.

⬤ Erros Clássicos dos Brasileiros
✗ "I tryed to call you."
✓ "I tried to call you."
TRY → TRIED (não "tryed"). Consoante antes do Y → Y vira IED. Mesma regra de STUDY → STUDIED.
✗ "She trys to be positive."
✓ "She tries to be positive."
HE / SHE / IT → TRIES (Y vira IES). "She TRIES to be positive." ✓ | "She trys" não existe.
✗ "I didn't tried the food."
✓ "I didn't try the food."
DIDN'T + BASE (try). "didn't tried" ✗ — DIDN'T já marca o passado; o verbo volta à base TRY.
✗ "Try to eating less sugar."
✓ "Try eating less sugar."
TRY + ING (experimentar) — sem TO. "Try EATING less sugar." ✓ | TRY TO + BASE (tentar): "Try TO eat less sugar." ✓ — nunca "try to eating".
✗ "Did you tried the new place?"
✓ "Did you try the new place?"
DID + BASE. "Did you TRY the new place?" ✓ — depois de DID o verbo volta sempre à base.
✗ "I will tried again."
✓ "I will try again."
WILL + BASE. "I will TRY again." ✓ — nunca TRIED depois de WILL.
◈ Quiz — TRY
Pergunta 1 de 12
Qual é o passado correto de TRY?
TRY → TRIED. Consoante antes do Y (R + Y): troca-se o Y por IED. Nunca "tryed" ou "tryied".
Pergunta 2 de 12
"Ele tenta se exercitar todo dia." → Presente com HE.
HE / SHE / IT → TRIES (Y vira IES). "He TRIES to exercise every day." ✓ | "trys" não existe.
Pergunta 3 de 12
Diferença entre "Try TO do" e "Try DOING":
"Try TO open the door" = tentei (mas talvez não consegui). "Try opening the window" = experimenta abrir (para ver se resolve). Significados diferentes!
Pergunta 4 de 12
"Você já experimentou sushi?" → Com HAVE YOU.
HAVE YOU + particípio. O particípio de TRY é TRIED. "Have you TRIED sushi?" ✓
Pergunta 5 de 12
"Posso experimentar esta calça?" → TRY ON.
TRY ON = experimentar roupa. "Can I try these pants ON?" ✓ — o ON pode vir depois do objeto. TRY OUT = testar algo novo, não roupa.
Pergunta 6 de 12
"Eu não tentei o prato novo." → Passado, negativo.
DIDN'T + BASE (try). "I didn't TRY the new dish." ✓ — DIDN'T já carrega o passado; verbo vai à base.
Pergunta 7 de 12
"Ela não tenta o suficiente." → Presente com SHE, negativo.
SHE → DOESN'T (não DON'T). Depois de DOESN'T: BASE sem mudança (try, não tries). "She DOESN'T TRY hard enough." ✓
Pergunta 8 de 12
"TRY OUT" significa:
TRY OUT = testar, dar uma chance a algo novo. "Let's try out the new restaurant." TRY ON = experimentar roupa/calçado.
Pergunta 9 de 12
"Você tentou ligar para ela?" → Passado, pergunta.
DID + BASE. "Did you TRY to call her?" ✓ — depois de DID o verbo vai sempre na base TRY, nunca TRIED.
Pergunta 10 de 12
"Tente não se preocupar." → Qual é a forma correta?
"Try NOT TO + BASE" = tente não fazer X. "Try not TO WORRY." ✓ — a negação (not) vai antes do TO, não depois.
Pergunta 11 de 12
"Vamos tentar de novo amanhã." → Com WILL.
WILL + BASE. "We will TRY again tomorrow." ✓ — base sempre depois de modais.
Pergunta 12 de 12
"Eles vão experimentar o restaurante novo." → GOING TO.
THEY → ARE going to + BASE. "They ARE going to TRY the new restaurant." ✓ — base TRY (não TRIED) depois de GOING TO.
