# SISTEMA UNIFICADO — JORNADA COMPLETA
# Inglês para Brasileiros — Do zero ao avançado em sequência inteligente

---

## FILOSOFIA DA TRILHA

A sequência não é alfabética nem por sistema.
É pedagógica — cada etapa prepara o terreno para a próxima.

PRINCÍPIOS:
1. Apresentar o conceito → mostrar no uso real → expandir
2. Gramática sempre depois do uso (não antes)
3. Vocabulário cresce junto com a estrutura
4. Revisão embutida — conteúdo anterior reaparece em contextos novos
5. Cada etapa tem no máximo 10-15 minutos de estudo
