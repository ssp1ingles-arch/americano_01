## O QUE É ÚNICO NO SISTEMA 13

O Sistema 13 é o **dicionário visual interativo** da plataforma. Cada palavra é apresentada com: pronunciação implícita (repetição), frase de exemplo curta e simples, e tradução. O formato "palavra → frase" é o mais eficiente para memorização de vocabulário no nível A1. Unifica dois vídeos com vocabulário complementar: objetos/comida/pessoas e frases funcionais/corpo humano.
